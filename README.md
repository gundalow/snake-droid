# Example App

This is an Android Snake game based on a 3D specification, adapted for a 2D top-down view in Jetpack Compose.

## Download APK
You can download the latest version of the game from our Netlify web server:

**[Download Snake Game APK](https://gundalow-snake.netlify.app/snake-game.apk)**

(Note: The URL depends on the `NETLIFY_SITE_ID` configured in GitHub Secrets. Replace with the actual URL once known.)
